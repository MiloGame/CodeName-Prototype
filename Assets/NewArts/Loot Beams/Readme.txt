About
Loot Beams is a combination of Line Renderer and Particle components.

Usage
Grab any Loot Beam prefab located in Loot Beams/Prefabs and then just drop to your scene.

Scale & Size
Every Beam created with Line Renderer component.
Here are two property for scale your beams - (Positions)Size and Width of Line Renderer.
- (Positions)Size is free for customize, by default all beams has 8 value. (You can also use Transform(Y) in child prefab (Line Renderers) for scale).
- Width of Line Renderer become locked with attached animator on parent object. If you want to customize Width just delete the Animator component or customize animations inside.
Particles can be re-sized all in by changing transform on child object (Particles).

Compatibility
Unity 2018 and later.
Standard, URP, LWRP and HDRP support.
Shaders constructed via Amplify Shader Editor.
Loot Beams require no scripts.
